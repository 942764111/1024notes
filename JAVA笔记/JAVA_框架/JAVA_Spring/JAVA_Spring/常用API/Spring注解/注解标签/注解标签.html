一、自动扫描的注解标记

1、@Component

  • 通用注解
  • 参数：
    @Component(beanID)默认beanid为小写开头的类名,也可以自定义beanid

2、@Named

  • 通用注解
  • 和Component一样

3、@Repository

  • 持久化层组件注解

4、@Service

  • 业务层组件注解

5、@Controller

  • 控制层组件注解

二、功能注解

1、@Scope("")

  • 指定某类的作用域

2、@PostConstruct

  • 指定初始化方法

3、@PreDestroy

  • 指定销毁方法

4、@Lazy(boolean)

- 指定类是否延时加载
